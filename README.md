# Kelvin Magumise — Portfolio

Personal portfolio website for Kelvin Magumise, Software Engineer & Entrepreneur.

**Live:** https://magumisekelvin.vercel.app

## Stack
- HTML5, CSS3, Vanilla JavaScript
- No frameworks, no build step — deploys anywhere instantly

## Features
- Custom animated cursor
- Smooth scroll reveal animations
- 3D tilt effect on project cards
- Animated hero with floating tech cards
- Fully responsive — mobile, tablet, desktop
- Contact form with success state
- Counter animations on stats
